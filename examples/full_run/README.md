# Full runs

Each folder is one complete run — a recipe executed on real input. Inside:

- `run.json` — the full board log: every step's input, output, and the specialist's own reasoning.
- `output/` — what it actually produced, **staged** for a human (labels, drafted replies, posts). Never sent.
- `README.md` — the recipe + the input in a line, and a link to watch it step by step.

Watch any of these click-through in the **[gallery](https://mtichikawa.github.io/workflow-engine/usecases/)**.
