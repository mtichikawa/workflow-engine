# refine-loop

**Recipe:** `refine`  ·  **Input:** why flaky tests erode team trust

Draft + self-grade + rewrite until it passes.

- `run.json` — the full board/run log (every step's input, output, and reasoning).
- `output/` — what it actually produced, staged for a human (never sent).
